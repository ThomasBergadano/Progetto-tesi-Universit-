Innanzitutto, per scaricare il progetto in locale ho dovuto scaricare react e node 
(ho usato Vite come strumento di compilazione/server di sviluppo locale),
scaricare l'SDK di Firebase e creare un file di configurazione, e creare una repository
GitHub, collegandola alla cartella in locale del mio progetto.

Per chi volesse testarlo, sarà necessario sia installare React che l'SDK di Firebase.

Per avviare il progetto, utilizzando un terminale (io ho usato quello di Visual Studio Code), 
bisogna:
   1. Entrare nella cartella "Client"
   2. Scrivere "npm run dev" per avviare l'applicazione: nonostante non si avvierà da sola, verrà
      mostrato il link in locale (es. http://localhost:80)
   3. Cliccando sull'indirizzo URL o digitandolo direttamente online, potrete navigare nell'app.


PS: la cartella "Server" e tutto quello che c'è dentro è inutilizzato, ma lasciato lì in caso
di aggiornamenti futuri e cambi di rotta.
